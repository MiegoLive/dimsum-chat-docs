# 砸砸乐自助更新

上传你已有的砸砸乐 `.ds` 文件，验证通过后即可下载最新版本。

<WidgetUpdater widget-slug="bonk" />


## 使用说明

1. 在点心Chat的「我的应用」中找到已安装的砸砸乐
2. 找到对应的 `.ds` 安装文件（通常在你下载时保存的位置）
3. 将 `.ds` 文件拖拽到上方区域，或点击选择文件
4. 验证通过后点击下载按钮获取最新版本
5. 在点心Chat「我的应用」中重新安装新版 `.ds` 文件即可

## 常见问题

**Q: 我找不到 .ds 文件怎么办？**

A: `.ds` 文件是你购买砸砸乐时下载的安装包。如果你找不到，可以从曾经的购买处重新获取下载。

**Q: 更新后需要重新配置吗？**

A: 砸砸乐的配置数据存储在浏览器 localStorage 中，更新 `.ds` 文件不会影响已有配置。但如果新版有重大变更，建议备份自定义规则。

**Q: 支持哪些版本的验证？**

A: 支持从 v1.0.0 到最新版的所有砸砸乐 `.ds` 文件验证，包括旧名称「抖音礼物砸脸」的文件。
