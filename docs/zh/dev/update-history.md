# 更新历史

- 取消默认自包含模式，默认数据存储位置改为 `%APPDATA%\MiegoLive\DimSumChat`

- 新增日志功能
- 新增 Websockets 会话列表显示
- 新增应用置顶与删除功能
- 新增若干功能……
